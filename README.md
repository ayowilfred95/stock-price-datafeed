## Project Overview

This project is designed to facilitate access to and adjustment of data feeds, which are essential for trading analysis and stock price monitoring. The meticulously adjusted data is configured on your system and seamlessly integrated into Perspective, thereby enhancing trading insights.

To offer traders a comprehensive view of all monitored trading strategies, multiple screens typically showcase a diverse range of live and historical data at their workstations. This holistic approach ensures that traders are well-informed and equipped to make informed decisions.

Given the substantial influx of information and data, it is crucial to visualize data clearly, taking into account UI/UX considerations. This clarity plays a pivotal role in empowering traders with effective tools to enhance their performance in the trading domain.

## Getting Started

To kickstart this project locally, follow these steps:

1. Navigate to the `backend` directory to focus on the server-side development.

2. Set up a virtual environment by executing the following command:

   ```bash
   python3 -m venv myenv
   ```

3. Activate the virtual environment by running the following command:

   ```bash
   source myenv/bin/activate
   ```

4. Proceed to install the required dependencies by executing the following command:

   ```bash
   pip install python-dateutil
   ```

5. Run the following command to start the backend/server application:

   ```bash
   python server3.py
   ```

6. Next, navigate back to the root folder and then into the `frontend` folder to start the frontend/client-side application.

7. Run the following command to install the dependencies:

   ```bash
   npm install
   ```

8. Finally, run the following command to start the application:

   ```bash
   npm start
   ```

Following these steps ensures a seamless setup and enables you to delve into the project development process effectively.
